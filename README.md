## Overview

OnArc is an **agentic freelance marketplace** built on Arc, designed to enable seamless global work and instant payments.

Instead of manually searching for talent or relying on trust-based agreements, OnArc uses **AI agents to match work intents with the best candidates**, while **programmable escrow contracts ensure secure and automatic payments**.

The result is a faster, more reliable way to coordinate and settle work — without intermediaries.

---

## Problem

Traditional freelance platforms suffer from:

- Slow and manual hiring processes

## MVP Scope

This MVP focuses on the **core flow of agentic hiring and settlement**.

---

### 🔹 1. User Profiles (Supply Layer)

Users can:

- Create a profile  
- Add skills and services  
- Upload portfolio / proof of work  
- Set pricing and availability  

This enables structured data for intelligent matching.

---

### 🔹 2. Job Intents (Demand Layer)

Clients can:

- Post job requests (intents)  
- Define budget, timeline, and requirements  

This forms the foundation for agent coordination.

---

### 🔹 3. AI Agent Matching

- Agents analyze job intent  
- Rank and select top 3 candidates  
- Present curated options to the user  

The user selects the final freelancer.

---

### 🔹 4. Escrow Contract (On Arc)

Once a freelancer is selected:

- A programmable contract is created  
- Client deposits stablecoins  
- Funds are locked as collateral  

---

### 🔹 5. Work Submission & Verification

- Freelancer submits work  
- System or agent verifies completion  

---

### 🔹 6. Instant Settlement

- Payment is released automatically  
- Settlement occurs in seconds  
- Finality is guaranteed
